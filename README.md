## Features

- **Responsive Design** - Mobile-friendly layout that adapts to all screen sizes
- **Smooth Navigation** - Active navigation highlighting and smooth scrolling
- **Project Portfolio** - Showcase of completed projects with descriptions
- **Contact Form** - Functional contact form with email validation
- **Social Media Links** - Direct links to social profiles
- **Animated Elements** - Scroll reveal effects and typed text animation

## Technology Stack

- **HTML5** - Semantic markup
- **CSS3** - Responsive styling with CSS Grid and Flexbox
- **JavaScript** - Vanilla JS for interactivity
- **External Libraries**:
  - Boxicons - Icon library
  - ScrollReveal - Scroll animations
  - Typed.js - Text typing animation

## How to Use

1. Clone or download the repository
2. Open `index.html` in a web browser
3. Navigate through the sections using the header navigation
4. Fill out the contact form to send an email

## Customization

### Update Personal Information
- Edit section content in `index.html`
- Update social media links in the Home section
- Modify the typed text strings in `js/Script.js`

### Modify Styles
- Edit color variables in `css/Style.css` (in the `:root` selector)
- Adjust responsive breakpoints for different screen sizes

### Add New Projects
- Add a new portfolio-box div in the portfolio section
- Include an image in `assets/images/`
- Update the portfolio grid if needed
